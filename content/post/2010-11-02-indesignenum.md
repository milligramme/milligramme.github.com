InDesignで二つのオブジェクトの属性の組み合わせを確認したいとき。例えば、「テキストフレームとグラフィックフレーム」の組み合わせとか、「ガイドが垂直と水平」とかの組み合わせか、「四角形と楕円形」の組み合わせなどをInDesignのEnumeration Valueを使って出来ないか試してみました。

![/images/2010/11/textframe_graphicframe.png](/images/2010/11/textframe_graphicframe.png)

TextFrameのContentTypeは1952412773、GraphicFrameのContentTypeは1735553140をEnum値（Number）として持っていて、合計で3687965913です。

```js
if (app.selection[0].contentType + app.selection[1].contentType === 3687965913 ) {
  //処理
};
```

って感じに、二つのオブジェクトの組み合わせ判定に使え簡素化できそう。

 *  @param {Object} obj Enumeration Value
 *  @param {Number} argu1 value 1
 *  @param {Number} argu2 value 2
 *  @returns {Array} array of combination
 */
function mat_check (obj, argu1, argu2) {
  var enum_arr=[];
  for (var i in obj) {
    enum_arr.push(i);  
  };
  res = [];
  for (var j=0;  j < enum_arr.length; j++){
    var base_key = parseInt(enum_arr[j]);
    for (var k=0; k < enum_arr.length; k++) {
      if (base_key + parseInt(enum_arr[k]) == argu1 + argu2){
        res.push (obj[ enum_arr[j] ] + "," + obj[ enum_arr[k] ]);
      };
    }
  };
  return res;
}
