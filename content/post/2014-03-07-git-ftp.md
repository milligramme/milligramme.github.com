### 初期化(git init)

`.git-ftp.log` ができる

```bash
$ git ftp init -u <user> -p - ftp://host.example.com/public_html
```

### アップロード(git push)

```bash
$ git ftp push -u <user> -p - ftp://host.example.com/public_html
```

`.git-ftp.log` にHEADのハッシュ値を書き込んでいく

`git rev-parse HEAD` と `cat .git-ftp.log` が同じになるはず
